Set up local

Insurance risk uses mostly Python for analytical procjets, data treatment and scripting.

Julia, R and other tools can be used if they fit the task better, for instance Julia in high performence computing and R for visualization. But Python is starting point by default.

Anaconda#

Start by downloading Anaconda, this can be installed without admin right, by choosing "local user" in the installation wizard.

This will install most packages you'll need to start working with analytics in Python, but a update could be neccessary and VS runtime is needed for many packages.

Choose "Anaconda prompt" from start menu and copy the following in the terminal

conda update conda
conda update --all
conda install -c conda-forge vs2015_runtime

Jupyter Notebooks#

Anaconda comes with Jupyter Notebook installed. Just choose "Jupyter Notebook" from the start menu.

It's also possible to install Jupyter Lab in order to get multiple notebooks or files within the same windows. Install by opening "Anaconda prompt" and type conda install -c conda-forge jupyterlab. Now run by type "Jupyter Lab" in the Anaconda prompt.

GIT#

Version controlling is a important part of every project, and all code should be managed in GitLab. So install GIT (admin rights not needed), and send request for GitLab user in serivce now.

VS Code#

When working with more demanding scripts, an regular text editor is often preffered. VS Code is a solid choice with a nice set of features!

Install the Pyhton and Python Docstring Generator extensions, and install pylint when asked so it automatically checks the code for errors.

Preview is also a nice extension, which gives continously updated preview when working with documents like markdown, HTML and RST.

R#

R and R Studio can be installed, but one need to contact Servce Desk in order to get them installed since they need admin rights.
