# Standard python packages
import os
import sys
# Other package imports
import numpy as np
import pandas as pd
# from matplotlib import pyplot as plt

Setup some global settings and configuration

project_root = os.path.abspath(os.path.join(os.getcwd(), os.pardir))
data_folder = os.path.join(project_root, 'data')
data_folder_raw = os.path.join(data_folder, 'raw')
src_folder = os.path.join(project_root, 'src')

Importing data

filename='policy_data.xlsx'
df=pd.read_excel(os.path.join(data_folder_raw,filename))
``
As we can see, the excelsheet contains birth day, gender and interest rate. The last two columns are the calculated premium intensity with coverage (here A for Alderspensjon/Age pension), contract start age and stop age. To more easily be able to calculate something from this data we need the data on flat format.
```python
df=df.drop('Unnamed: 0', axis=1)
df_flat=df.melt(id_vars = df.columns[:3], value_vars=['A6499','A6799'], var_name='Coverage',value_name='Premium')
print(df_flat.head().to_markdown())

df_flat.groupby('Coverage')['Premium'].count().plot.bar(x='Coverage', y='Premium', stacked=True)
